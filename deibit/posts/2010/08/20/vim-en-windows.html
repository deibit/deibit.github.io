Si instalamos Vim en Windows y queremos exportar nuestro Vimrc (desde Linux o Mac):

- Averiguar donde guardar el vimrc:

introducimos :echo $HOME

ahí deberemos guardar el vimrc que en Windows deberá nombrase a _vimrc

- Fuentes:

para cambiar el tipo de letra:

set guifont=Consolas:h10

Donde h es el tamaño de la fuente. Si necesitamos espacios en blanco para describir el nombre:

set guifont=Andale_mono:h9

solo debemos sustituirlos por guión bajo ‘_’
