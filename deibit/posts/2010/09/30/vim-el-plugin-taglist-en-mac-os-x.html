El muy útil plugin para Vim TagList necesita de ‘Exhuberant tags’ para funcionar. El ctags que trae Mac cuando se instala XCode y compañía presenta ciertos problemas cuando es usado por TagList (no funciona correctamente).

Lo primero es instalar ‘ctags’ desde MacPort:

sudo port install ctags

Tras la instalación debemos incorporar a nuestro .vimrc la siguiente cadena:

let Tlist_Ctags_Cmd=’/opt/local/bin/ctags’

Con esto le decimos al plugin que cuando quiera usar ctags éste se encuentra en la localización que le hemos añadido. Así podrá usar el ctags de MacPort que es el que necesita.
