vim: Filtrar texto (varios trucos)

David García

2010-04-28 21:08

Source

* Antes una utilísima función para “grepear” partes del texto hacia un registro:

:g/patron/y A

Se guardará en el registro “a

Básicamente es: g = grep /patron_a_encontrar/ y = yank A = append

* Leer la salida de un comando hacia el editor:

:r!ls

r = read ! = ejecutar_un_comando ls = comando_ls

:-r!ls

Lo mismo que el anterior pero el caracter ‘-’ al principio hará que el texto se inserte antes de la linea actual.

* Usar un programa externo para filtrar texto:

:%!sort

% = todo_el_fichero ! = ejecutar_un_comando sort = comando_sort

:10,20!sort

Lo mismo que el anterior pero solo de la linea 10 a la 20

V
(seleccionar lineas)
!sort

Es lo mismo que los anteriores pero usando el modo visual linea
