vim: Exportar a html...

David García

Si tenemos texto que queremos exportar a HTML, Vim nos provee de un plugin estándar para hacerlo.

Seleccionamos el rango (que obviamente puede ser todo el archivo) e introducimos el comando ‘TOhtml’

El siguiente fragmento (aunque exigiría algunos retoques) está exportado con el mismo.

#include

int main()

{

    int ( *p )() = NULL;

    (*p)();

    return 0;

}
