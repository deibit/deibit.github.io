David García

Comments

Source

Title says it all:

[sourcecode language="cpp"]
// $clang++ –std=c++0x –stdlib=libc++ snippet.cpp -o snippet

#include <vector>
#include <iostream>

using namespace std;

int main()
{
auto vv = vector<vector<int>>(); //1
auto v = vector<int>(); //2
v.push_back(1);
v.push_back(2);
for(int z:v) //3
{
cout << z << endl;
}
return 0;
}
[/sourcecode]

1.- No more angle bracket mess
2.- auto type inference
3.- new for (…the win!)
