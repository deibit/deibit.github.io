Fast Metasploit set-up (linux)

David García

2012-09-24 16:16

Source

Every time I have to install Metasploit in a new machine I need to dive into zillions of pages about setting up MSF (for Metasploit Framework).
Setting it up is not complicated but when it comes to connect postgres…things tends to be harder.

Before, make sure to install

So first, get MSF from github repository:

git clone https://github.com/rapid7/metasploit-framework.git metasploit

install ruby dependencies:

gem install pg

$ sudo -u postgres createuser –superuser msf
$ sudo -u postgres psql
postgres=# \password msf
(I choose msf too)
postgres=# createdb msfdb
postgres=# \q

Now on MSF:

db_connect msf:msf@localhost/msfdb
